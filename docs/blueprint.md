# **App Name**: Digital Property Insights

## Core Features:

- Homepage: A visually appealing landing page that introduces the website's purpose and offers a clear call to action.
- ValuVista Page: Details the ValuVista application, including key features, benefits, and a call-to-action for a demo.
- RenoScope Page: Details the RenoScope application, including a step-by-step guide, target audience, and a call-to-action to get started.
- About Us Page: Provides background information on the company's story, mission, and values.
- Contact Us Page: Provides contact details and a contact form for visitors to send inquiries.

## Style Guidelines:

- Primary color: Blues for a trustworthy and technological feel.
- Secondary color: Whites to create a clean and professional backdrop.
- Tertiary color: Light grays.
- Body and headline font: Clean sans-serif font for a modern appearance.
- Implement a grid-based layout for orderly arrangement of content.
